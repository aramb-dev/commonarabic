Commonly Used Arabic Phrases by Abdur-Rahman Bilal

This repo includes PNG & Photoshop files used to make these stickers.


Where can I use them?

You can use them anywhere! All you need to do is upload a .png file to wherever you want them. They are published officially on Telegram and some third-party WhatsApp sticker apps.

  1. Telegram Sticker => click here
  2. Sticker Maker Studio Sticker => click here
  3. Sticker Maker Sticker => code: NV4XK1


This sticker pack is still being updated often.
